This plugin creates Imprint, Privacy, Accessibility pages for Wordpress CMS.

== Description ==
Terms of Service ( TOS ) plugin let webmaster show in an easy way the required pages
Imprint, Privacy and Accessibility by creating a custom menu and templates with information from 
webmaster portal (WMP) or by filling forms in the admin menu.

Admin can modify TOS information from settings menu > TOS.    

== Installation ==
1. Upload the contact-form-plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin using the ‘Plugins’ menu in your WordPress admin panel.
3. You can adjust the necessary settings using your WordPress admin panel in “Settings” > “TOS”.
4. Click the "Update info from Web Master Portal (WMP)" button to get all information if the domain is registered with the RRZE otherwise fill the form with the information.
5. For the FAU themes, TOS menu will be activated automatically. For general themes, TOS menu will be created but not activated, the webmaster can place the menu in any defined location.

== Frequently Asked Questions ==

= What does this plugin do? =
The principle behind this plugin is to enable Imprint, Privacy and Accessibility pages automatically, allow
the webmaster to modify the content.

= Is this plugin only for FAU webpages? =
TOS plugin is for all web pages made in Wordpress that want to show to the final user the terms of service
is a clear way.
